# Kalaba Mobile Demo

Отдельный репозиторий для мобильной версии Kalaba Demo.

## Как открыть через GitHub Pages

1. Создать новый репозиторий: `kalaba-mobile`
2. Загрузить все файлы из этого архива в корень репозитория
3. Открыть Settings → Pages
4. Source: Deploy from a branch
5. Branch: main
6. Folder: / (root)
7. Save

После публикации сайт будет доступен по адресу:

https://gamaleyandrey-create.github.io/kalaba-mobile/

## Важно

- `index.html` должен лежать в корне репозитория
- `styles.css` и `app.js` тоже должны лежать в корне
- Это мобильная демо-версия
