# Kalaba Mobile Other Category

Добавлено:
- категория «Другое» в главной, поиске и создании заявки;
- при выборе «Другое» появляется поле ручного ввода услуги;
- поле становится обязательным;
- ручной текст услуги передаётся в параметры заявки/поиска;
- переводы: русский, украинский, испанский, английский.

Загрузить поверх файлов в `kalaba-mobile`.
