# Kalaba Mobile Active Links

Мобильная версия Kalaba Demo с максимально активными переходами.

Активировано:
- поиск;
- категории;
- карточки компаний;
- кнопки заявок;
- карта;
- статистика;
- предложения;
- чат;
- сделки;
- оплата;
- отзывы;
- CRM/кабинеты;
- строки таблиц.
